# pecaIncluidaNaOrdemDeServico

## Descrição

Indica que uma peça foi adicionada à Ordem de Serviço.

## Emissor

```text
oficina-os-service
```

## Tópico canônico

```text
oficina.os.peca-incluida-na-ordem-de-servico
```

## Consumidores potenciais

```text
oficina-billing-service
oficina-execution-service
```
