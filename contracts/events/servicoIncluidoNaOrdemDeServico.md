# servicoIncluidoNaOrdemDeServico

## Descrição

Indica que um serviço foi adicionado à Ordem de Serviço.

## Emissor

```text
oficina-os-service
```

## Tópico canônico

```text
oficina.os.servico-incluido-na-ordem-de-servico
```

## Consumidores potenciais

```text
oficina-billing-service
oficina-execution-service
```
