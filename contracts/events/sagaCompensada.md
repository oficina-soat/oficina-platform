# sagaCompensada

## Descrição

Indica que uma transação distribuída executou ações compensatórias após falha em uma das etapas.

## Emissor

```text
oficina-os-service
```

## Tópico canônico

```text
oficina.saga.saga-compensada
```

## Consumidores potenciais

```text
oficina-billing-service
oficina-execution-service
```
